1. Open the Preferences menu within the application.

4. The software will scan the folder and update your sound list automatically.

## 🛠 Solving Common Issues

If you do not hear sound, check these settings:

*   Verify your speakers or headphones connect correctly to your computer.
*   Ensure the volume sliders in the software remain in the middle of their range.
*   Check the MIDI channel settings if you use an external keyboard.
*   Make sure no other programs are currently using your audio interface.

If the application runs slowly, close other heavy programs like web browsers or video editors. Large sound libraries require fast read speeds, so keep your files on a Solid State Drive (SSD) if possible.
